#CorFinGateway G77 application directory structure

```
appsrv/          - Application server deployment.
buildSrc/        - Common build components for the MAH.
configDataSets/  - Used for population of unit test data.
environments/    - Environment definitons.
gateway/         - DDL and aptitude projects.
gradle/          - Gradle dependenices.
lib/             - Common JAR files used for deployment purposes.
quality/         - Code quality common subcomponent for QC of deployed DDL/DML.
```
